Временно:

для запуска тачки на линукс хосте в одном терминале запустите:
docker-compose down -v
docker-compose up --build

*запускайте с корневой папки проекта

Во втором терминале, так же с корневой папки проекта:

npm run dev

Если выдает ошибку, то скорее всего не установлен npm:

npm install




В самом браузере для чека страницы:

http://localhost:5173/


для чека статуса:

http://localhost:8000/health
